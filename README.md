# Spendlet

A fully local, privacy-first budgeting app designed to run entirely within your browser.

- **Zero Wait & Zero Subscriptions**: No sign-ups needed.
- **Total Data Privacy**: Uses `LocalStorage` and local AI (`tesseract.js`) so your bank statements never leave your device.
- **Mobile First**: Sleek, mobile optimized design.

## Features
- Smart Paste: Copy rows directly from your bank and parse them automatically.
- Screenshot Scanning: Upload mobile screenshot receipts for immediate local parsing.
- Monthly Breakdowns: Automatic visual aggregations of spend by category.
